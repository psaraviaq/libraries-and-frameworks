Grid System

A responsive systems that can layout and align content. To start with, everything must be inside a "container". Then, we use "row" and "col" classes.

You can just use the "col" class for every column to be the same width, or to occupy the space left in a row.


    <div class="container">
      <div class="row">
        <div class="col">
          <div>abc</div>
        </div>
        <div class="col">
          <div>123</div>
        </div>
        <div class="col">
          <div>xyz</div>
        </div>
      </div>
    </div>
abc
123
xyz

You can also add numbers from "1 to 12", or even "auto" to resize to its content. If we exceed the 12 grid, the rest will just go to the next row.

Breakpoints can be added for different layouts depending of the screen. They can be "sm", "md", "lg", "xl", "xxl", or just nothing (which is "xs"). If no "xs" class is defined, then the elements will occupy the whole row at that width.

Keep in mind that smaller classes apply also to larger screens, unless overridden by larger classes.


    ...
    <div class="col-sm-6   col-md-3     col-xl">...</div>
    <div class="col-6      col-sm-auto  col-md-3">... </div>
    <div class="col-sm     col-md-6     col-xl">...</div>
    ...

    ...
    <div class="row row-cols-4">
      <div class="col">...</div>
      ... (12 times)
